# Elias Nymander — Portfolio

Personal portfolio site for Elias Nymander, an Information Systems MSc
graduate targeting junior AI/GenAI developer and AI consulting roles in
Finland. Single-page, bilingual (English/Finnish), covering his background,
projects, work experience, education, and skills.

Built with Vite + React + Tailwind CSS. No backend — fully static, deployed
on Vercel.
